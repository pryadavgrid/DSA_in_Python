1. Constant Time : BIG O of 1 → O(1)
    when no loop, no recursion every line run in constant way
    exp. 

        x = 5
        print(x)
        if x == 5 : 
            print("Hello")
        else:
            print("Welcome")

    1. Here no any loop mean (jo h vo print ho raha h) 
    2. it mean code does not depend on input size 
    3. in this case we say that this code time complexity is O(1)


2. linear Time : 
    in the linear time we convert our code into in a linear mathamatical equation
    exp.

        n = 5                   → 1 unit
        print(n)                → 1 unit
        for i in range(n):      → n unit
            print(i)            → n unit

            = 1 unit + 1 unit + n unit + n unit
            = 1 + 1 + n + n
            = 2 + 2n            → linear equation

        1. when code depend on single loop then time complexity is O(n)
        2. Note : when multiple loop run independent then time complexity also O(n)
        
    exp.
        n = 5                   → 1 unit
        print(n)                → 1 unit
        for i in range(n):      → n unit
            print(i)            → n unit
        
        for i in range(n):      → n unit
            print(i)            → n unit

            = 1 unit + 1 unit + n unit + n unit + n unit + n unit
            = 1 + 1 + n + n + n + n
            = 2 + 4n            → linear equation

3. Quadreatic Time : 

    when a loop execute inside a loop then we say thet time complexity is O(n^2)

    exp.
        n = 5                   → 1 unit
        print(n)                → 1 unit
        for i in range(n):      → n unit
            for j in range(n):  → n unit
                print(i)        → n^2 unit
        
        for i in range(n):      → n unit
            print(i)            → n unit

            = 1 unit + 1 unit + n unit + n^2 unit + n unit + n unit
            = 1 + 1 + n + n + n^2 + n + n
            = 2 + 4n + n^2       → Quadreatic equation

4. Cubic time complexity :
    when a loop execute inside a loop then we say thet time complexity is O(n^2)

    exp.
        n = 5                       → 1 unit
        print(n)                    → 1 unit
        for i in range(n):          → n unit
            for j in range(n):      → n unit
                for k in range(n):  → n unit
                    print(k)        → n^3 unit
        
        for i in range(n):          → n unit
            print(i)                → n unit

            = 1 unit + 1 unit + n unit + n unit + n^3 unit + n unit + n unit
            = 1 + 1 + n + n + n+ n^2 + n + n
            = 2 + 4n + n^3       → Cubic equation


5. Logarithmic Time :
    when code and value after every execution become sepratly decrese then time complexity is  O(log n). -(order of log n)
    when code and value after every execution become sepratly increase then time complexity is O(log n)

    exp 1.

        print(n)            

        while (n >= 1):     
            print(n)        
            n = n//2

    exp 2.

        n = 1
